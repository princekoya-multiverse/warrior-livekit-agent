# Warrior Doctrine — Reference Knowledge

This file is loaded into the agent's context as supporting knowledge. Keep it
concise; shorter context = faster + cheaper inference.

## The Warrior OS Chain
Health → Clarity → Decisions → Wealth → Freedom → Purpose

- **Health gives you Clarity.** A tired, inflamed body makes foggy decisions.
- **Clarity improves Decisions.** Clear thinking compounds.
- **Better Decisions build Wealth.** Wealth is the residue of good decisions.
- **Wealth creates Freedom.** Freedom is optionality — the ability to say no.
- **Freedom lets you live with Purpose.** Purpose is the point of all of it.

## Health Foundations (in order)
1. **Sleep** — 7–8 hours, dark and cool room. Fix this before anything else.
2. **Movement** — 30 minutes daily that raises the heart rate.
3. **Remove one drain** — usually a food, a habit, or a person.

## Wealth System
Problem → Validation → Minimum Offer → First Revenue
- Don't build a product first. Find three people with the problem.
- If they'd pay today, you have a business. If not, iterate.

## Decision Frameworks
- Reversible decisions: decide fast, move.
- Irreversible decisions: slow down, gather proof.
- Default to the option that increases optionality.

## Tech & Web3 Stance
- Self-host where it matters. Avoid vendor lock-in.
- Decentralization is a tool for sovereignty, not a religion.
- Choose boring, proven tools over hype.
